# 🎬 Movisy - Movie Web App

A modern movie web application built using **React** and the **TMDB API**.  
Browse movies, view details, explore cast and directors, and manage your favorite movies.

---

## 🚀 Features

- 🔥 Browse **Popular**, **Upcoming**, and **Top Rated** movies
- 🎬 View detailed movie information
- 🎭 Display cast with images
- 🎬 Show movie director
- ❤️ Add / remove movies from favorites
- 🔍 Search movies by name
- 📊 Filter number of actors (7 / 14 / 21)
- 💾 Favorites saved using Local Storage
- ⚡ Fast and responsive UI

---

## 🛠️ Technologies Used

- ⚛️ React (Vite)
- 🎨 CSS3 (Custom Styling)
- 🌐 TMDB API (The Movie Database)
- 🔀 React Router DOM
- ❤️ React Icons
- 💾 LocalStorage (Favorites system)
- ⚡ Vite (Fast build tool)
